# Cloud Drive 

## Project Overview
Cloud Drive is a secure and user-friendly platform for managing and storing files in the cloud. This repository contains the API Toolkit for the Cloud Drive frontend, built with modern web technologies.

## Features
- **Authentication**: Secure login system with client-side validation.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Theming**: Light and dark mode support.
- **Reusable Components**: Modular UI components built with React and TailwindCSS.

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm (v8 or higher)

### Installation
1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```
2. Navigate to the project directory:
   ```sh
   cd api-toolkit-main
   ```
3. Install dependencies:
   ```sh
   npm install
   ```

### Development
Start the development server:
```sh
npm run dev
```

### Build
To create a production build:
```sh
npm run build
```

## Deployment
This project is ready for deployment on platforms like Vercel. Ensure environment variables are configured in a `.env` file.
